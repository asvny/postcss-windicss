# postcss-windicss

[![NPM version](https://img.shields.io/npm/v/postcss-windicss?color=a1b858&label=)](https://www.npmjs.com/package/postcss-windicss)

> Expiremental.

## Install

```bash
npm i -D postcss-windicss postcss
```

```js
// postcss.config.js
module.exports = {
  plugins: {
    'postcss-windicss': { /* ... */ },
  },
}
```

```css
/* main.css */
@windicss;
```
